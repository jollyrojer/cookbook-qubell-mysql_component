MySQL Cookbook
==============
Installs and configure MySQL-server.
Allow run MySQL management tasks.

Platform
--------
- Ubuntu 10.04, 12.04 (x32/x64)
- CentOS 5, 6 (x64)

Cookbooks
---------
Requires Opscode's cookbooks:
- build-essential '1.4.2'
- database '1.5.3'
- mysql '3.0.13'
- openssl '1.1.1'
